- Responsive design for all devices
- Interactive project showcase
- Contact form integration
- Modern UI with smooth animations

- Formspree for contact form

## Sections

- Hero with profile introduction
- About & Education
- Experience & Achievements  
- Skills & Technologies
- Projects showcase
- Contact form
